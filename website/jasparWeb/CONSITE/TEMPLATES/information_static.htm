Information about Consite and associated programs

What is Consite

Consite is a web-based tool for detecting transcription factor binding sites in genomic sequences using phylogenetic footprinting. Two orthologous genomic sequences are aligned, and transcription factor binding sites are only reported for those regions in the alignment which transcend a certain treshold of conservation

Consite also provide means for selecting transcription factor binding profiles to scan sequences with by information content, species, domain class.

In reporting, Consite provides a wide number of options for displaying the result of the scans

Features

Input options

Consite accepts one or two genomic sequences as input, with an option to include cDNA and/or exon postions.

Transcription factor selection options

Transcription factor binding profiles can be chosen by

  • minimal specificity
  • name
  • species and species supergroup(s)
  • domain class

A list of selected transcription factors can also be downloaded for later upload and analysis.

Consite reports options:

Three different views can be chosen:

Graphical view showing

  • alignments graphically with transcription factor binding site locations indicated,
  • percentage identity plots with coding regions highlighted if a cDNA sequence was supplied.

Alignment view showing

  • detailed alignment and transcription factor binding sites within the alignment
  • a highlighting of conservwed regions

Table view showing

  • all reported transcription factor binding sites in tabular form
  • indication of coding/non-coding region

All views can be modified by user by entering

  • range of displayed sequence relative to either input sequence
  • conservation cutoff
  • window size
  • min. information cutoff

Transcription factor binding site profiles used in Consite

The transcription factor binding site profiles used in Consite are all high-quality, non-redundant experimental binding site studies (mostly SELEX-protocol), from multicellular eukaryotes. All binding profiles have been thouroughly checked and binding sites have been realigned using Gibbs sampler programs.

The database will be made downloadable in the near future.
