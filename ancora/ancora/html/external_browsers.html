Browse HCNEs in external genome browsers

The Ancora genome browser provides the most flexible way to explore the Ancora data. However, it is also useful to view the data in other browsers where it can be compared to annotations not present on this site.

UCSC Genome Browser

All files in our downloads section can be uploaded as custom tracks to the UCSC Genome Browser. Please visit the downloads section for more details.

Ensembl

Much of the Ancora data is accessible through the DAS protcol for exchanging sequence annotations. Ensembl can display sequence annotations provided in this way. We have created a form by which HCNE data easily can be added to ContigView in Ensembl. Once you have added a track to Ensembl in this way, Ensembl will remember it as part of your preferences, so that you do not have to add it again until your Ensembl preferences expire. Proceed to add HCNE tracks to Ensembl.
