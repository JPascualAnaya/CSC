Download data

Data files are available for HCNE locations and HCNE density curves. The files are in BED and WIG format, respectively. All files are compressed with the gzip program.

HCNE location files are named

HCNE_asm1_asm2_Xpc_Ycol.bed.gz

where asm1 and asm2 are identifiers for the compared genome assemblies, X is the percent identity threshold used for HCNE detection and Y is the number of alignment columns (window size) over which percent identity is computed (see Methods for details).

HCNE density files are similarly named

HCNE_density_asm1_asm2_Xpc_Ycol.wig.gz

The coordinates in the files refer to assembly asm1.

All files can be used as custom tracks in the UCSC Genome Browser. It is not neccessary to download the files in order to use them as custom tracks. Simply paste the URL for the file of interest into the "add custom tracks" form on the UCSC Genome Browser web site. To copy the URL for a file, right-click on the filename with and select "Copy Shortcut" or "Copy Link Location". If you are using a Mac with a single mouse button, hold down Ctrl while clicking on the filename. Make sure you add tracks to the correct genome assembly; its identifier should match asm1.

Please select the genome assembly that you wish to download data for:

  • Human March 2006 assembly (NCBI 36 / hg18)
  • Mouse July 2007 assembly (NCBI 37 / mm9)
  • Mouse February 2006 assembly (NCBI 36 / mm8)
  • Zebrafish July 2007 assembly (Zv7 / danRer5)
  • Zebrafish March 2006 assembly (Zv6 / danRer4)
  • Tetraodon February 2004 assembly (V7 / tetNig1)
  • D. melanogaster April 2006 assembly (BDGP 5 / dm3)
  • C. elegans January 2007 assembly (WS170 / ce4)

Before using this data in a large-scale analysis, please note that usage restrictions may apply to the underlying genome sequences.
